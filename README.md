 # JavaFX-soplayer

配有详尽注释

**描述**

采用 JavaFX 组件完成的一款小巧、界面精美的本地音乐播放器，使用.ini 文件记录应用设置信息。 




**已实现模块：**

- 支持拖动添加文件 

- 支持系统托盘和全屏控制 

- 界面简洁而精美且支持自定义背景 

- 支持播放的音乐格式：.mp3 文件、.wav 文件、.aac 文件 

- 支持解析歌词并展示（.lrc 文件） 

- 支持解析歌词文件（缩略图、专辑、时长等） 

- 支持频谱图动效 

- 支持歌单列表控制 

**运行环境：**

 Java(TM) SE Runtime Environment (build 1.8.0_241b07)
